# Налаштування Gemini API для чату

## Файл `.env` вже створено

Переконайтеся, що в файлі `.env` є:

```env
VITE_GEMINI_API_KEY=ваш_gemini_api_ключ_тут
```

## Як отримати API ключ:

1. Перейдіть на https://aistudio.google.com/apikey
2. Створіть новий API ключ
3. Скопіюйте його та вставте в файл `.env`

## Модель Gemini 2.5 Flash:

- ✅ Найкраща модель за співвідношенням ціна/продуктивність
- ✅ 1,048,576 токенів на вхід
- ✅ 65,536 токенів на вихід
- ✅ Підтримка тексту, зображень, відео, аудіо
- ✅ Thinking mode для складних запитань

## Важливо:

- Файл `.env` вже додано в `.gitignore` і не буде комітитися
- Після оновлення файлу перезапустіть dev сервер: `npm run dev`
- Для production на Vercel додайте змінну `VITE_GEMINI_API_KEY` в налаштуваннях проекту
